Position property has 4 options

  1. Static
  2. Relative
  3. Absolute
  4. fixed

      5. Static

      static is the default value for the position property

      Top, bottom, left, and right will NOT work with static.

      This element will show up, top to bottom, left to right. Nothing special, and it can't be moved

      Relative

      A relative position is the same as static (top to bottom< left to right), except you can move it with top, bottom, left and right.

      IF you move it, it will sit on top of the other stuff and the rest of the page will think its still where static
